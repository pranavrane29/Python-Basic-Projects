## 📌 Key Projects

### 🔢 Smart Calculator & Data Manager
A comprehensive menu-driven program combining:
- Basic arithmetic with exception handling
- Scientific calculations using the `math` module
- Random number generation using the `random` module
- Persistent result history using dictionaries
- Modular design with separate functions for each feature

### 🎓 Student Management System
A dictionary-based CRUD application supporting:
- Adding student records with roll number as key
- Searching students by roll number using `.get()`
- Displaying all records with `.items()` iteration

### 👨‍💼 Employee Record System
An OOP-based solution featuring:
- `Employee` class with tuple-based detail storage
- Dictionary mapping employee IDs to objects
- Method-driven display of all records
